# 隐私政策

欢迎您使用【平台名称】。我们深知个人信息及数据安全对您的重要性，并将依法采取相应安全保护措施。为帮助您了解我们如何收集、使用、存储、共享和保护您的信息，以及您享有的相关权利，我们制定本《隐私政策》。

请您在使用本平台服务前，认真阅读并充分理解本政策全部内容。您在注册、登录、访问或使用本平台服务时，即表示您已知悉并同意我们按照本政策处理您的相关信息。

## 第一条 适用范围

1. 本政策适用于【运营主体名称】在提供【平台名称】网站、控制台、API 网关及相关产品和服务过程中，对用户信息进行处理的行为。
2. 本政策不适用于第三方独立向您提供的服务。若您通过本平台调用第三方模型、支付、认证、通知、云服务或其他外部服务，该等第三方可能依据其自身隐私政策或服务规则处理您的信息，建议您另行阅读相关第三方文件。

## 第二条 我们收集的信息类型

为实现服务功能、保障系统安全、完成计费结算和满足合规要求，我们可能收集以下信息：

### 一、账户与身份信息

包括但不限于：

- 用户名、显示名称、邮箱地址；
- 密码哈希值及账户登录凭据；
- 账户角色、状态、分组、偏好设置；
- 第三方登录或绑定信息，如 GitHub、Discord、OIDC、Telegram、WeChat 等平台账号标识；
- 访问令牌、账户识别信息及关联信息。

### 二、安全验证与认证信息

在您启用安全验证功能时，我们可能处理：

- 双重验证相关信息；
- Passkey / WebAuthn 凭证信息，包括凭证标识、公钥材料、验证状态、设备能力标记、最近使用时间等；
- 风险验证、登录校验、异常登录判断所必需的信息。

### 三、API Token 与权限配置信息

包括但不限于：

- API Token 标识、名称、创建时间、过期时间、状态；
- Token 额度、调用限制、模型限制、分组策略；
- Token 允许访问的 IP 范围等权限配置。

### 四、交易与支付信息

当您进行充值、订阅或支付时，我们可能收集：

- 订单编号、交易流水号、支付方式、金额、币种、订单状态、完成时间；
- 支付回调信息、支付渠道返回信息、订阅计划与权益信息；
- 第三方支付客户标识及完成交易所必需的关联信息。

### 五、调用记录与日志信息

当您使用平台 API 或控制台时，我们可能记录：

- 请求时间、请求路径、请求 ID、响应状态、耗时；
- 使用的模型、渠道、Token 名称、调用次数、额度消耗；
- Prompt Tokens、Completion Tokens、统计信息、重试信息；
- 错误代码、错误类型、错误信息、故障排查所需信息；
- 您主动启用相关设置时记录的 IP 地址。

### 六、您主动提交的内容

为实现具体功能，您可能向平台提交：

- 文本输入、提示词、消息内容、参数配置；
- 图片、音频、文件、文件链接或其他附件；
- 与模型调用、格式转换、内容检测、文件下载、缓存处理有关的数据。

### 七、设备、会话与访问环境信息

在您访问网站或控制台时，我们可能收集：

- 浏览器类型、设备类型、操作系统、语言设置、访问时间；
- Cookie、Session、本地存储信息；
- 页面访问行为和站点统计数据；
- 若部署者启用统计分析工具，则可能产生访问分析数据。

## 第三条 我们如何使用您的信息

我们收集和使用信息，主要用于以下目的：

1. 完成用户注册、登录、身份识别、账户管理及安全验证。
2. 提供统一 AI 网关服务，包括请求接收、鉴权、模型路由、转发、重试、结果返回及兼容处理。
3. 实现 API Token 管理、权限控制、IP 限制、调用审计和风控治理。
4. 进行费用计算、额度扣减、账单生成、充值入账、订阅开通、订单管理和财务对账。
5. 生成统计报表、使用分析、控制台展示、系统监控和运营分析。
6. 检测异常行为、处理故障、优化性能、排查风险、保障平台和用户安全。
7. 向您发送验证码、通知、预警、账单、订单状态或其他必要服务信息。
8. 满足法律法规、监管要求、争议处理、投诉处理、司法协助、税务审计和合规留痕等义务。

## 第四条 关于 AI 请求内容及第三方上游处理

1. 本平台属于统一 AI 网关与调用管理平台。您通过平台提交的提示词、消息内容、参数、文件、图片、音频、附件或链接，可能根据您的调用目标、路由策略或所选模型，被发送给相应第三方上游服务商进行处理。
2. 您理解并同意，为实现文本生成、图像生成、音频处理、向量化、重排序、实时交互等服务，平台有必要在技术所需范围内传输相关请求数据。
3. 不同上游服务商可能基于其自身服务规则、隐私政策和合规要求，对相关数据进行处理、留存、审查或限制。平台无法完全控制第三方的处理行为。
4. 如您使用本平台接入第三方上游模型，建议您同时阅读相应第三方的服务条款和隐私政策，并根据自身业务需要评估是否适用。

## 第五条 日志、缓存与存储机制

1. 为保障接口稳定、支持重试机制、提高系统性能、完成计费或处理文件请求，平台可能在请求处理期间使用内存缓存、磁盘临时缓存、数据库缓存或 Redis 缓存等机制。
2. 临时请求体缓存、文件下载缓存及相关技术缓存通常用于单次请求处理、转发、解析、重试或性能优化，并在技术流程结束后按系统机制清理。
3. 平台将保存账户资料、Token 数据、订单数据、订阅数据、调用日志、错误日志及统计聚合数据，以支持服务提供、故障排查、审计和运营分析。
4. 日志信息一般以调用元数据、计费信息、错误信息和审计信息为主，用于系统管理、账务处理和安全控制。

## 第六条 我们共享、转让或披露信息的情形

### 一、共享

我们仅在以下情形下共享您的信息：

- 为实现服务功能，与上游模型服务商、云服务商、支付机构、身份认证服务商、消息通知服务商共享必要信息；
- 为完成支付、订阅、认证、内容处理、文件下载、通知投递、合规校验等目的，在必要范围内共享；
- 根据法律法规、监管要求、司法或行政机关依法提出的要求进行提供；
- 在获得您的授权同意或您主动选择相关功能的情况下进行共享。

### 二、转让

未经您的明确同意，我们不会向其他公司、组织或个人转让您的个人信息。但在发生合并、分立、资产转让、重组、股权交易、破产清算等情形时，如涉及个人信息转移，我们将要求接收方继续依法履行保护义务。

### 三、公开披露

除以下情形外，我们不会公开披露您的信息：

- 已取得您的单独同意；
- 依据法律法规、监管规则或有权机关依法要求必须披露；
- 为保护国家安全、公共安全、您或他人的生命财产安全，在紧急情况下依法进行披露。

## 第七条 Cookie、Session 与本地存储

1. 为保障登录状态、维持会话连续性、保存界面设置、提升交互体验和增强安全性，平台可能使用 Cookie、Session、本地存储等技术。
2. 相关技术通常用于身份识别、会话保持、配置保存、风险控制和访问分析。
3. 您可以根据浏览器设置管理相关功能，但部分服务在关闭相关技术后可能无法正常使用。

## 第八条 信息保存期限

1. 我们将在实现本政策所述目的所必需的最短期限内保存您的信息，法律法规另有规定或监管另有要求的除外。
2. 在您注销账户、终止使用服务或达到保存期限后，我们将依据法律规定、合规要求及系统处理逻辑，对相关信息进行删除、匿名化处理，或继续保留法律要求必须保存的部分数据。
3. 对于涉及财务、交易、税务、争议处理、安全审计、投诉处理的记录，我们可能在合理或法定期限内继续保存。

## 第九条 信息安全保护措施

1. 我们采取合理可行的技术与管理措施保护您的信息安全，包括但不限于权限控制、认证机制、访问隔离、日志留痕、加密或脱敏处理、安全审计等。
2. 我们会持续改进安全能力，尽力防止信息遭受未经授权访问、泄露、篡改、损毁或丢失。
3. 但请您理解，互联网环境并非绝对安全，任何技术措施均无法保证百分之百安全。请您妥善保管账户和密钥，避免在不安全环境中使用服务。
4. 如发生或可能发生个人信息安全事件，我们将根据法律法规要求采取补救措施，并在必要时向您告知事件影响、处置方式和建议措施。

在适用法律规定范围内，您对您的信息享有相应权利，包括但不限于：

- 查询、访问与复制您的账户信息、订单信息、调用记录等；

如您需行使相关权利，可通过本政策载明的联系方式与我们联系。我们将在核实您的身份后，在合理期限内处理您的请求。

2. 如您为未成年人，请在监护人指导下阅读并决定是否同意本政策，并在监护人同意的前提下使用本平台服务。
